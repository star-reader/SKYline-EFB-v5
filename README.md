# 项目部署
使用```npm install```安装依赖

使用```npm run serve```启动开发环境

使用```npm run build```进行项目打包

# 说明
使用前请【务必】阅读“写在前面”部分

请将crypto中的加密密钥和偏移量替换为自己项目的，相应接口为https://api.skylineflyleague.cn/efb_api_v2/crypto/sign

请注意产品使用合法性，部署打包上线时请确保阅读navigraph相应开发者文档并遵守其开发规定
